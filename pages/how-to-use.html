How to Use

Simple directions for the visitation tools, media library, and giving page

Use the site with confidence

This page is for church workers and users who need quick directions. Choose the area you need, then follow the steps below. Each section explains what the page is for, which button to use, and what to expect next.

Visitation

Use visitation updates, reports, new mover visits, and printable card tools.

Photo Repository

Sign in, create folders, upload files, move items, and download media.

Giving Page

Give by GCash, PayPal, mail, drop-off, or by scanning the giving QR code.

How to Use the Visitation Page

The visitation page is the starting place for follow-up work and visitation records.

1

Open the page

Go to the Visitation page and choose the task you need. The page includes new mover visits, visitation updates, reports, and card creation tools.

2

Choose the correct tool

Use Visit New Move Ins to see new mover information, Visitation Card Update to submit a report, Visitation Report to review saved cards, and Visitation Card Creation to make printable cards from a spreadsheet.

3

Submit accurate details

Enter names, addresses, contact information, and notes carefully. This helps the church keep reliable follow-up records for future visits.

4

Review before finishing

Before saving or generating a file, check spellings, dates, and house information. Correct entries make reports and printable cards more useful.

Helpful reminders

  • Use the card update page when you already have visitation details and need to record them online.
  • Use the report page when you want to search or review saved visitation entries.
  • Use the card creation page when you have an Excel or CSV list and want printable PDF cards.
Open Visitation Open Card Update Open Report

How to Use the Photo Repository

The Harvest Media Library stores photos, videos, documents, and ministry media files.

1

Sign in first

Enter your email and password on the Media Library login form. After sign-in, the library will open and show your folders and files.

2

Go to the right folder

Use the folder list on the left side to open the correct category. The breadcrumb at the top helps you know where you are.

3

Upload or organize media

Click Upload Files or drag files into the upload area. You can also create folders, move files to another folder, or delete items you no longer need.

4

Open file details when needed

Click a file to view details, download it, or move it to another folder. This is the easiest way to organize the library after uploading.

Best practices

  • Name folders clearly so other workers can find files quickly.
  • Upload files into the correct ministry or event folder before sharing them.
  • Double-check before deleting a folder because removing a folder also removes its contents.
Open Media Library

How to Use the Giving Page

The giving page shows the available ways to support the ministry online or in person.

1

Choose your giving method

The page offers GCash, PayPal, mail-in giving, and drop-off giving. Pick the option that works best for you.

2

Use GCash or PayPal online

For GCash, scan the QR code shown on the page. For PayPal, click the PayPal button and complete the gift on the secure PayPal page.

3

Use mail or drop-off if preferred

The page also lists the church mailing address and office hours, so you can send or deliver your offering in person.

4

Ask for help if needed

If you have questions about giving statements or records, use the contact information at the bottom of the page.

Quick links on the giving page

  • The GCash card includes a scannable QR code.
  • The PayPal card opens a secure donation form in a new tab.
  • The giving QR code section helps users open the giving page from a mobile device.
Open Giving Page

Need more help?

If you are unsure which page to use, start with the main Visitation, Media Library, or Give page and compare the button names with the directions above. If the information you need is not listed, contact the church office for assistance.
